# СК ВЕКТОР — Лендинг

Одностраничный сайт строительной компании «ВЕКТОР». Возведение зданий под ключ за 90 дней: склады, цеха, фермы, аренда. Санкт-Петербург и Москва.

## Стек

- HTML5
- CSS3 (встроенный, без препроцессоров)
- Vanilla JavaScript
- Шрифты: Inter, Montserrat (Google Fonts)

## Структура

```
vektor/
├── index.html      # Единственная страница со встроенными стилями и скриптами
├── .gitignore
├── .editorconfig
└── README.md
```

## Запуск локально

Проект статический — достаточно открыть `index.html` в браузере.

Для удобной разработки с автоперезагрузкой можно использовать любой live-сервер, например:

```bash
# через npx (если установлен Node.js)
npx serve .

# или Live Server в VS Code (расширение ritwickdey.LiveServer)
```

## Деплой

Подходит любой статический хостинг: GitHub Pages, Netlify, Vercel, Cloudflare Pages, Beget, обычный хостинг по FTP.
